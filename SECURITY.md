# Security Policy

## Reporting Security Issues

Please report security issues responsibly.

If GitHub Security Advisories are available for this repository, use:

```text
https://github.com/aia11111/Media_Fetch_aia/security/advisories/new
```

If advisories are not available, open a minimal public issue asking for a private security contact. Do not include exploit details, private URLs, cookies, tokens, credentials, or account data in a public issue.

## Responsible Disclosure

Please include:

- A short description of the issue
- Affected files or workflows
- Reproduction steps, if safe to share privately
- Potential impact
- Suggested mitigation, if known

## Prohibited Areas

This project does not support bypassing DRM, paywalls, login restrictions, authentication, or platform access controls.

Please do not submit reports or pull requests that add, request, or demonstrate circumvention of protected access systems.

## Legal And Ethical Use

Media Fetch AIA is intended only for media that the user owns, has permission to download, or is legally allowed to archive. Users are responsible for complying with each platform's terms of service and applicable laws.
