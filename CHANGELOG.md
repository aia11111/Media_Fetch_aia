# Changelog

## v0.1.0 - 2026-06-03

Initial public project baseline.

### Added

- Windows desktop GUI built with customtkinter.
- Media download workflows built around yt-dlp.
- gallery-dl fallback support for Instagram workflows.
- URL support paths for YouTube, Instagram, Threads, and Naver Blog.
- Video and audio download modes.
- Resolution selection and subtitle option support.
- Queue-based downloads with progress display.
- Local download history with thumbnails and file access.
- Duplicate file handling policies: ask, auto-rename, overwrite, and skip.
- Windows URL protocol registration for `yg-download://`.
- PyInstaller one-dir build flow.
- Public README, MIT license, contribution guide, security policy, and minimal CI syntax check.
